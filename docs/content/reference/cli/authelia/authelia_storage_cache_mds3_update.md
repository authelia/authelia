## authelia storage cache mds3 update

Update WebAuthn MDS3 cache storage

### Synopsis

Update WebAuthn MDS3 cache storage.

This subcommand allows updating of the WebAuthn MDS3 cache storage.

```
authelia storage cache mds3 update [flags]
```

### Examples

```
authelia storage cache mds3 update
```
